NanamiOS Version 1.0 [Codename: "Reimu"] Build 20

System Requirements:
- CD drive
- Intel i386 or better (Pentium 4 recommended)
- 64 megabytes of RAM
- 4 GB hard drive
- A display

Warnings:
- Only one driver is provided in this version. It is very primitive and has almost no driver support, apart from a keyboard driver.
- If you want to run this on physical hardware, do so at your own risk. Nothing should go wrong, but I recommend you boot off of a CD.

Known Issues:
- Command Line doesn't exist.
- Only one driver exists.
- No backspace key
- Driver glitch
- Issue with boot screen.
- Glitch with the mouse driver causes a problem with the text display.

Patch Notes:
- Nothing!

What's new or changed:
